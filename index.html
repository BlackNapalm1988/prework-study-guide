HTML

  • HTML defines the foundation and structure of a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The H1 thru H6 heading tags format text on the page.
  • HTML Color Codes

CSS

  • CSS is used to style a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The HTML link tag allows you to link a CSS file.
  • The "." indicator specifies a class selector, defining HTML property assignments.
  • MDN Web Docs on CSS

GitHub

  • GitHub is a cloud-based version control software for developers.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git branch --delete branch-name: deletes an existing branch
  • git add -a: adds all changed files of the current branch to the staging area.
  • git commit -m "message": commits all staged files to the repository.
  • git pull origin main: incorporates changes from a remote repository into the current branch.
  • git push origin 'branch-name': push up all changes to the repository.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Unlike Python, you will need to add a semi-colon at the end of each line.
  • Also, boolean values are written in lower case true/false unlike Python True/False
  • Script tag will need to be placed before the closing body tag.
